# Hospital-SQL-Project
This Hospital SQL Project is a database management system designed to streamline hospital operations. It organizes and manages data for patients, doctors, departments, diagnoses, medications, and treatments. The database includes tables such as Patient, Doctor, Worker, Department, Diagnosis, Medication, and Tests.

With this project, users can perform SQL queries to retrieve information, such as total salary expenditure, doctor assignments, patient diagnoses, and department statistics. The system is robust and versatile, ideal for understanding and implementing hospital data workflows.

To get started, clone the repository, set up the database, and execute the provided SQL scripts.
